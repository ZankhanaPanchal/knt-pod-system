Where does this data go?

Every POD submitted here — details and photos — is written automatically to a Google Sheet that acts as dispatch's live dashboard. It's shared only with dispatch, not public.
  • Each row is one ticket. Photos are saved to a linked Google Drive folder, with clickable links right in the sheet.
  • To find a specific POD, use the Sheet's search or filter by ticket number or driver name.
  • If a driver updates a POD later, the existing row is updated in place and a "Last Updated" column shows when.
  • Ask dispatch for the Sheet link if you need to check on a submission.
